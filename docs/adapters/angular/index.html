Angular Adapter

The Angular adapter (@angular-architects/native-federation) provides first-class Native Federation support for Angular — a builder that wraps the Angular CLI, schematics to scaffold hosts and remotes, and helpers for Angular-specific concerns like locale data.

Documentation in progress. Overview content for the Angular adapter will land here. In the meantime, the sub-pages below cover specific Angular scenarios.

Angular-Specific Guides

  • Angular I18N — full internationalization support with Native Federation
  • Angular Localization — locale data loading and the ignoreUnusedDeps feature
