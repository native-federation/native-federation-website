esbuild Adapter

Applies to

The @softarc/native-federation-esbuild package is a thin, framework-agnostic adapter that plugs the core builder into esbuild. It powers the React reference example and can be used with any stack where you drive your own build.

Documentation in progress. This section is a placeholder for the esbuild adapter docs — including React-specific setup, fileReplacements for CommonJS libraries, and compensateExports.

Installation

npm i @softarc/native-federation
npm i @softarc/native-federation-esbuild
