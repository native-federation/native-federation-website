# Angular Architecture Workshop

> Learn Native Federation, Micro Frontends, and advanced Angular architecture patterns in an interactive online workshop by Angular Architects.

Go beyond the basics with an interactive, 100% online workshop covering Native Federation, Micro Frontends, and enterprise-grade Angular architecture.

## What You'll Learn

The Angular Architecture Workshop by Angular Architects covers Native Federation and far more. It provides different architectural options and alternatives, explaining the consequences of each approach so you can make informed decisions for your projects.

### Topics Include

- Micro Frontend architecture with Native Federation
- Strategic domain design for Angular applications
- Module boundaries and dependency management
- Advanced state management patterns
- Performance optimization strategies
- Testing strategies for federated applications
- Production deployment and CI/CD pipelines
- Migration strategies from monoliths

## Format

- **100% online** — join from anywhere
- **Interactive** — hands-on exercises and live Q&A
- **Expert-led** — taught by the creators of Native Federation
